Clustering


Clustering is an unsupervised machine learning technique. In clustering the goal is to divide a set of data points into clusters or groups with some clustering algorithm. As mentioned, clustering is unsupervised, which means that we don't have access to the ground truth labels for the data points during training the model or running the clustering algorithm.

There are many ways to approach the problem of clustering data points into groups. The choice of approach can depend on the application. Some common approaches are hierarchical, centroid based and density based clustering algorithms. Hierarchical clustering algorithms start either with one large cluster and divide it into smaller clusters based on some rule, or they start with each data point in their own cluster and combine clusters until all data points are in one cluster or the desired number of clusters have been reached. Centroid based clustering like k-means clustering use centroids around which the clusters are formed. Density based clustering algorithms like DBSCAN detect areas where data points are densely packed and uses the density to determine which data point belongs into which cluster.
